# Static data & demos

This folder is served as static files by Nginx (see `deploy/nginx/my_homepage.conf`).

## Data

- `data/profile.json`: Personal info used by the homepage.

## Demos

- `data/apps/player/`: A tiny HTML audio player demo. Put your own audio file as `data/apps/player/audio.mp3`.
- `data/apps/player/index.html`: A simple launcher page (choose which demo to open).
- `data/apps/game/`: A tiny HTML mini-game demo.
